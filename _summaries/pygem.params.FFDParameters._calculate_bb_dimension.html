pygem.params.FFDParameters._calculate_bb_dimension¶

static FFDParameters._calculate_bb_dimension(shape, tol=1e-06, triangulate=False, triangulate_tol=0.1)[source]¶

Calculate dimensions (minima and maxima) of a box bounding the

Parameters:
  • shape (TopoDS_Shape) – or a subclass such as TopoDS_Face the shape to compute the bounding box from
  • tol (float) – tolerance of the computed bounding box
  • triangulate (bool) –

    Should shape be triangulated before the boudning box is created.

    If True only the dimensions of the bb will take into account every part of the shape (also not visible)

    If False only the visible part is taken into account

    *See build_bounding_box()

  • triangulate_tol (float) – tolerance of triangulation (size of created triangles)
Returns:

coordinates of minima and maxima along XYZ
Return type:

tuple
