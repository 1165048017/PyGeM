pygem.unvhandler.UnvHandler.parse¶

UnvHandler.parse(filename)[source]¶

Method to parse the file filename. It returns a matrix with all the coordinates. It reads only the section 2411 of the unv files and it assumes there are only triangles.

Parameters:filename (string) – name of the input file.
Returns:mesh_points: it is a n_points-by-3 matrix containing the coordinates of the points of the mesh.
Return type:numpy.ndarray
