Module with all the plotter functions for the different file extensions.

plot_opts(save_fig=True, fig_type='.png', fig_dir='figures')[source]

This method sets some options for the plots.

Parameters:
  • save_fig (bool) – flag to save or not the image into a directory called figures. Default is True.
  • fig_type (str) – the file extension of the file to save. Default is ‘.png’
Returns:

opts: the chosen options. The keys in the dictionary are fig_type, save_fig, and font. The font is a dictionary that sets the font properties of the figures.
Return type:

dict
plot_stl(filename, output_name=None, opts=None)[source]

Method to plot an stl file.

Parameters:
  • filename (str) – the stl filename you want to plot.
  • output_name (str) – filename of the output file. If None it takes the filename and changes the extension accordingly with the opts.
  • opts (dict) – dictionary with the options for the plot. It is set by plot_opts()
