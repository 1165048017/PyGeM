class IgesHandler[source]

Bases: pygem.nurbshandler.NurbsHandler

Iges file handler class

Variables:
  • infile (string) – name of the input file to be processed.
  • outfile (string) – name of the output file where to write in.
  • extensions (list) – list of extensions of the input/output files. It is equal to [‘.iges’, ‘.igs’].
  • control_point_position (list) – index of the first NURBS control point (or pole) of each face of the iges file.
  • tolerance (float) – tolerance for the construction of the faces and wires in the write function. Default value is 1e-6.
  • shape (TopoDS_Shape) – shape meant for modification.

Warning

  • For non trivial geometries it could be necessary to increase the tolerance. Linking edges into a single wire and then trimming the surface with the wire can be hard for the software, especially when the starting CAD has not been made for analysis but for design purposes.
