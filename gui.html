class Gui[source]

Bases: object

The class for the Graphic Unit Interface.

Variables:
  • filename_geometry (string) – input geometry to be morphed.
  • filename_parameters (string) – input parameters file for FFD.
  • check_var_dump_orig (int) – dump or not the original FFD lattice.
  • check_var_dump_morphed (int) – dump or not the morphed FFD lattice.
  • outfilename (string) – name of the output file geometry. The extension of the file is set automatically equal to the on of input file ‘filename_geometry’.
  • outfilename_lattice_orig (string) – name of the dumped file for the original lattice. The extension of the file is set automatically equal to ‘.vtk’.
  • outfilename_lattice_mod (string) – name of the dumped file for the morphed lattice. The extension of the file is set automatically equal to ‘.vtk’.
  • root (tkinter.Tk) – main window object of the GUI.
  • print_geometry_path (string) – geometry path to be printed close to the ‘pick geometry’ button.
  • print_parameter_path (string) – parameters file path to be printed close to the ‘pick parameters’ button.
  • label_geo (tkinter.Label) – label related to ‘print_geometry_path’.
  • label_params (tkinter.Label) – label related to ‘print_parameters_path’.
  • url (string) – url of the github page of PyGeM.
  • logo_panel (tkinter.Canvas) – canvas for PyGeM logo.
  • img (tkinter.PhotoImage) – PyGeM logo.
  • orig_geo_frame (tkinter.Frame) – frame for plotting of the original geometry.
  • mod_geo_frame (tkinter.Frame) – frame for plotting of the final geometry.
_chose_geometry()[source]

The private method explores the file system and allows to select the wanted geometry. Up to now, you can select only IGES, OpenFOAM, STL, UNV or VTK geometry file.

_chose_parameters()[source]

The private method explores the file system and allows to select the wanted parameters file. It visualizes only .prm files.

_goto_website()[source]

The private method opens the PyGeM main page on github. It is used for info about PyGeM in the menu.

_main()[source]

The private method inizializes and visualizes the window.

_run_simulation()[source]

The private method runs the geometrical morphing.

start()[source]

This method inizializes and starts the GUI.
