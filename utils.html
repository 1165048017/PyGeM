_write_vtk_box(box_points, filename, dimensions)[source]

Private method that writes a vtk file containing FFD control points.

Parameters:
  • box_points (numpy.ndarray) – coordinates of the FFD control points.
  • filename (string) – name of the output file.
  • dimensions (list) – dimension of the lattice in (x, y, z) directions.

Warning

If you want to visualize in paraview the inner points, you have to slice the lattice because paraview does not visualize them automatically even in the wireframe visualization.

plot_rbf_control_points(parameters, save_fig=False)[source]

Method to plot the control points of a RBFParameters class. It is possible to save the resulting figure.

Parameters:
  • parameters (RBFParameters) – parameters of the Radial Basis Functions interpolation.
  • save_fig (bool) – a flag to save the figure in png or not. If True the plot is not shown and the figure is saved with the name ‘RBF_control_points.png’. The default value is False.
write_bounding_box(parameters, outfile, write_deformed=True)[source]

Method that writes a vtk file containing the FFD lattice. This method allows to visualize where the FFD control points are located before the geometrical morphing. If the write_deformed flag is set to True the method writes out the deformed lattice, otherwise it writes one the original undeformed lattice.

Parameters:
  • parameters (FFDParameters) – parameters of the Free Form Deformation.
  • outfile (string) – name of the output file.
  • write_deformed (bool) – flag to write the original or modified FFD control lattice. The default is set to True.
Example: 
>>> import pygem.utils as ut
>>> import pygem.params as pars
>>> import numpy as np

>>> params = pars.FFDParameters()
>>> params.read_parameters(filename='tests/test_datasets/parameters_test_ffd_sphere.prm')
>>> ut.write_bounding_box(params, 'tests/test_datasets/box_test_sphere.vtk')
